# Interpret

Projet personnel pour le fun, la passion...
Ebauche d'un interpréteur avec une syntaxe proche de celle du C.

Syntaxe actuellement acceptée :

```
int main (int args) {
    int a, b = 2, c;
    for (i in 0 .. 10) {
        $printi (i); //syscall
    }
    while (1) {
        $printi (a);
    }
}
```
